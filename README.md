## Project Description
RNA sequencing (RNA-seq) is a high-throughput sequencing technique used to study gene expression by determining the quantity and sequences of RNA molecules present in a biological sample. This repository contains a structured workflow for RNA-seq data analysis, including data management, quality control, reference preparation, alignment, and gene expression
